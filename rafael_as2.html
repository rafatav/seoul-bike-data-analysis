Introduo

Seoul Bike Data o dataset a ser analisado nesse trabalho. O dataset contm a quantidade de bicicletas alugadas na cidade de Seul, capital da Coria do Sul, entre os anos de 2017 e 2018. O objetivo desse trabalho o de analisar e usar as informaes mais relevantes retiradas do dataset para realizar uma anlise preditiva prevendo a quantidade de bicicletas que sero alugadas com base em variveis climticas e temporais. O problema de regresso supervisionada, pois queremos prever um valor numrico (quantidade de bicicletas alugadas).

Usamos mtricas como MAE, RMSE e R2 para avaliar a performance preditiva de cada modelo:

In[10]:
def avaliar_modelo(modelo, X_test, y_test, nome):
    pred = modelo.predict(X_test)
    print(f"\nModelo: {nome}")
    print("MAE:", mean_absolute_error(y_test, pred))
    print("RMSE:", root_mean_squared_error(y_test, pred))    
    print("R2:", r2_score(y_test, pred))
    return pred

pred_lr = avaliar_modelo(modelo_lr, X_test, y_test, "Linear Regression")
pred_rf = avaliar_modelo(modelo_rf, X_test, y_test, "Random Forest")
